# Lead Communications Dashboard

Real-time dashboard for viewing all lead communications including:
- Twilio call logs
- SMS conversations
- Retell AI transcripts with summaries

## Live Dashboard

**Main Dashboard:** [View Communications](https://yodastardust.github.io/lead-communications-dashboard/)

**Marketing Presentation:** [View Slides](https://yodastardust.github.io/lead-communications-dashboard/marketing_presentation.html)

## Features

- Search and filter leads by name, phone, or address
- View complete communication timeline per lead
- Full Retell AI transcripts with sentiment analysis
- SMS conversation history
- Detailed call logs with duration and status

## Supabase Integration

When connected to Supabase, the dashboard updates in real-time as new data comes in from the n8n workflow.
